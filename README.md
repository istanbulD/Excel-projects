Bu depo, ileri düzey Microsoft Excel becerilerimi sergilediğim çeşitli veri analizi projelerini içermektedir. Projelerde kullanılan temel yetkinlikler şunlardır:

İleri Düzey Fonksiyonlar: INDEX, MATCH, VLOOKUP ve dinamik formüllerle veri yönetimi.

Veri Görselleştirme: Pivot Tablolar ve etkileşimli grafikler (Dashboards) ile anlamlı raporlar oluşturma.

Pratik Uygulamalar: Finansal takip, müşteri veri yönetimi ve kişisel sağlık takibi (Fitness Tracker) projeleri.
